# DesktopSwitcher
# 🖱️ 虚拟桌面切换工具

一个轻量级的 Windows 工具，支持通过鼠标中键快速切换虚拟桌面。

## ✨ 功能简介

- 📌 **中键触发切换**：点击鼠标中键即可触发 `Ctrl + Win + ←` 快捷键，自动向左切换虚拟桌面。
- ⚙️ **后台运行**：启动后常驻后台，无界面干扰。
- 🚀 **即开即用**：无需安装，双击运行即可使用。

## 🖥️ 使用场景

- 多任务办公时快速切换不同桌面
- 提高多窗口管理效率
- 适合习惯使用鼠标操作的用户

## 🧩 安装与使用

1. 下载并解压本项目文件
2. 双击运行 `DesktopSwitcher.exe`
3. 点击鼠标中键，即可向左切换虚拟桌面

---

## 🔄 添加到开机启动项（推荐）

让工具在每次开机时自动运行：

### 方法一：手动添加到「启动」文件夹

1. 按下 `Win + R` 打开运行窗口
2. 输入 `shell:startup` 并回车，打开启动文件夹
3. 将 `DesktopSwitcher.exe` 的快捷方式拖入该文件夹

> ✅ 下次开机时，程序将自动启动并在后台运行。


## 🧪 技术说明

- 使用 AutoHotkey 编写脚本并打包为 EXE
- 快捷键模拟：`^#Left`（Ctrl + Win + ←）
